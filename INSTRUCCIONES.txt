Instrucciones para generar la base de datos en firebase

1-Crear un nuevo proyecto en Firebase

2-Logeate en firebase: firebase login  y luego inicia sesión en firebase: firebase init

3-Sigue las instrucciones en firebase para realizar registrar tu app web y luego reemplaza
los valores de: const firebaseConfig = {
                             apiKey: " ",
                             authDomain: " ",
                             projectId: " ",
                             storageBucket: " ",
                             messagingSenderId: " ",
                             appId: " ",
                             measurementId: " ",
                           };

    que se encuentra dentro de main.ts en la raiz del proyecto.
    Encontraras los valores de firebase en settings > general

4-Vincula el proyecto local a un espacio de trabajo en firebase usando: ng build y firebase deploy

5-Desde tu cuenta en Firebase podras dar rol de administrador a usuarios que hayan creado una cuenta en el sitio web y con ello
 podran acceder a la ventana de administrador dentro de la página.

6-Desde la ventana de administrador puedes crear nuevos productos manualmente o cargar el archivo excel que viene dentro de
 este proyecto: "productos_juguetes_30.xlsx". También puedes crear tus propias colecciones de productos o agregarles descuentos.
 Para ver un ejemplo completo visita el mio en: tienda-juguetes.web.app
